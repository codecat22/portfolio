Furhat Culinary Instructor

Ioana Cotutiu | August Heddini | Sophia Hettich | Siyana Ivanova

Using the robot Furhat, we created an interactive cooking assistant/ instructor for beginner home cooks. Furhat walks users through recipes, allowing them to ask questions and revisit steps along the way.

The project involved designing and implementing the interaction, then conducting an user evaluation with two groups of participants - one using the robot to cook, one not. Results showed that while the robot did not significantly improve efficiency, the users were engaged in the experience and often reported increased enjoyment.

The report detailing our project and pilot study can be found here, and a video showing off the Furhat Instructor is available to watch here.
